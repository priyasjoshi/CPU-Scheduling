CPU Scheduling
Features:
------------------------------------------------------------------------------------------
- The program is implemented in C++ language to implement given three cpu scheduling to create a scheduling framework.
- User needs to enter the commandline arguments in order to instruct the program for type of input and output.
- There are three algorithms are implemented FIFO, SRJF(preemptive)and Round Robin with user inputted time slice.
- Input can be a file input and or randomly produced by program.
- Similarly, output can be displayed in the default or verbose mode.

Bugs:
--------------------------------------------------------------------------------------------
None

Limitations:
---------------------------------------------------------------------------------------------

- Input file format is not comma-seperated. Following is the format of the input file.

process_id arrival_time burst_time
0 9 25
1 17 10
2 32 7
3 35 20
